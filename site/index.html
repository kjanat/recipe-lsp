recipe-lsp in the browser

A CodeMirror editor wired to the recipe-lsp language server — compiled to WebAssembly and run in a Web Worker, no backend. The server is imported live from jsDelivr, the editor from esm.sh. Highlighting, squiggles, hover and completion all come from the LSP.
